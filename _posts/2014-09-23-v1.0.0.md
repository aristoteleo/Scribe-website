Monocle is now distributed through the Bioconductor Project. Upgrading to the current release is recommended for all users.  

You can upgrade Monocle as follows:

{% highlight R %}
> source("http://bioconductor.org/biocLite.R") 
> biocLite()
{% endhighlight %}
